  4. EWI part 4:

    • Glashow-Weinberg-Salam Theory
    • Quarks and the Standard Model
    • Optical Theorem and propagators

  5. EWI part 5:

    • Discrete symmetries
    • Mass matrix of the quarks
    • Decays
    • Strong CP problem
    • Pion decay
    • Scattering $e^+e^-\to f\bar{f}$
